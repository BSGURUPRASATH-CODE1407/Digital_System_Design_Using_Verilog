# Number Systems

---

## 1. Introduction

Number Systems are a way of representing and expressing numbers using a specific base or radix.

In Digital System Design, number systems define how data is stored and processed inside digital hardware. Computers do not use decimal internally; they use binary and related systems.

---

## 2. Why This Topic Matters

Number systems are the foundation of all digital electronics.

They are important because:

- Computers operate using binary data (0 and 1)
- All arithmetic and logic operations depend on number representation
- Memory addresses and data storage use binary/hexadecimal formats
- Digital circuits are designed based on binary logic levels

Without understanding number systems, it is impossible to understand how a computer works internally.

---

## 3. Theory

A number system is defined by its base (radix), which determines the number of unique digits it uses.

### Common Number Systems:

- **Decimal (Base 10):** Uses digits 0–9  
- **Binary (Base 2):** Uses digits 0 and 1  
- **Octal (Base 8):** Uses digits 0–7  
- **Hexadecimal (Base 16):** Uses digits 0–9 and A–F  

### Key Concepts:

- Each digit has a positional value based on powers of the base
- The rightmost digit has the least significance (LSB)
- The leftmost digit has the most significance (MSB)

### Example:

Binary number: 1011  
= (1×2³) + (0×2²) + (1×2¹) + (1×2⁰)  
= 8 + 0 + 2 + 1 = 11 (Decimal)

---

## 4. Applications

Number systems are used in:

- Digital circuit design (logic levels 0 and 1)
- CPU operations and arithmetic logic units (ALU)
- Memory addressing and data storage
- Communication protocols and encoding systems
- Verilog and hardware description languages
