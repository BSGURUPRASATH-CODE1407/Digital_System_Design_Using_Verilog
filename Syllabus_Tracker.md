# Digital System Design – Syllabus Tracker

## About This Tracker

I created this tracker to monitor my progress while learning Digital System Design and Verilog HDL.

For every topic, my goal is not just to complete the theory but also to:

* Understand the underlying concepts
* Draw and analyze circuit diagrams
* Write Verilog code
* Create testbenches
* Verify outputs through simulation
* Document interview questions
* Understand practical applications

A topic will be marked as completed only after all of the above tasks are finished.
