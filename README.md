# Digital_System_Design_Using_Verilog
Complete Digital System Design learning repository covering Number Systems, Boolean Algebra, Combinational &amp; Sequential Circuits, Memory Devices, FPGA fundamentals, Verilog implementations, testbenches, waveforms, interview questions, and practical projects.

Welcome to my Digital System Design learning repository.

I created this repository to document my journey of learning Digital System Design from the ground up through practical implementation using Verilog HDL. Rather than focusing only on theory, this repository emphasizes hands-on learning through RTL coding, simulation, waveform analysis, and digital design projects.

The repository covers the complete Digital System Design syllabus, including Number Systems, Boolean Algebra, Combinational Logic Circuits, Sequential Circuits, Memory Devices, Logic Families, and FPGA fundamentals.

For each topic, I aim to provide:

* Clear theory notes
* Design procedures
* Truth tables and circuit diagrams
* Verilog implementations
* Testbenches
* Simulation waveforms
* Interview questions
* Common mistakes
* Real-world applications
* Practice problems

This repository serves three purposes:

1. To strengthen my understanding of Digital System Design.
2. To build a professional portfolio for VLSI, RTL Design, FPGA, and Digital Design roles.
3. To create a structured learning resource that can help students and juniors learn the subject more effectively.

Whether you are studying Digital System Design for academics, placements, interviews, or VLSI careers, I hope this repository helps you learn both the theory and practical aspects of digital hardware design.
